SynergyUI

A clean, modern, fully-featured UI library for Roblox. Built for performance, customization, and ease of use.

local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

⚡ One-line install • Config saving • Flags support • No dependencies

Getting Started

Load the Library

local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

Create Your First Window

local window = SynergyUI:CreateWindow({
    Title = "My Hub",
    AccentColor = Color3.fromRGB(138, 141, 168),
    ConfigFile = "config.json",
    ToggleKey = Enum.KeyCode.RightShift,
    Parent = nil,
    CloseOnEscape = false
})

Window Configuration

Tabs

window:CreateTab(name, iconUrl?) → returns tab object with control constructors.

local main = window:CreateTab("Main")
local settings = window:CreateTab("Settings", "rbxassetid://123456789")

Controls Reference

Toggle Flag

Boolean switch with state saving.

main:CreateToggle({Name="ESP", CurrentValue=true, Flag="ESP", Callback=function(state)end})

Slider Flag

Numeric slider with increment.

main:CreateSlider({Name="Speed", Range={16,250}, Increment=1, CurrentValue=50, Flag="WalkSpeed"})

Button

Clickable action button.

main:CreateButton({Name="Greet", Callback=function() SynergyUI:Notify("Hello!",2) end})

Dropdown Flag

Selectable list.

local dd = main:CreateDropdown({Name="Mode", Options={"Normal","Aggressive"}, Flag="Mode"})

Checklist Flag

Multi‑selection list.

main:CreateChecklist({Name="Features", Options={"Aimbot","ESP"}, Flag="Features"})

TextInput

Free text field.

main:CreateTextInput({Name="Message", Placeholder="Enter..."})

NumberInput

Numeric input field.

main:CreateNumberInput({Name="Amount", CurrentValue=10})

Keybind Flag

Key capture with press detection.

main:CreateKeybind({Name="Ultimate", CurrentKeybind="F", Flag="UltKey"})

ColorPicker

RGB color selector.

main:CreateColorPicker({Name="ESP Color", Color=Color3.fromRGB(0,255,0)})

RadioGroup

Exclusive option buttons.

main:CreateRadioGroup({Name="Theme", Options={"Dark","Light"}})

Paragraph & Section

main:CreateSection("Info")
main:CreateParagraph({Title="Info", Content="text"})

Complete Example

-- SynergyUI Advanced Demo – Full featured with flags
local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

local window = SynergyUI:CreateWindow({
    Title = "SynergyUI • Complete Demo",
    AccentColor = Color3.fromRGB(138,141,168),
    ConfigFile = "demo_config.json",
    ToggleKey = Enum.KeyCode.RightShift,
    CloseOnEscape = true
})

local main = window:CreateTab("Controls")
local advanced = window:CreateTab("Advanced")

main:CreateToggle({Name="Master Switch", Flag="Master", CurrentValue=false, Callback=function(val) print(val) end})
main:CreateButton({Name="Notify Me", Callback=function() SynergyUI:Notify("Button clicked!",2) end})
main:CreateSlider({Name="Volume", Range={0,100}, Increment=5, CurrentValue=50, Flag="Volume"})
local dd = main:CreateDropdown({Name="Game Mode", Options={"Casual","Competitive"}, Flag="GameMode"})
local cl = main:CreateChecklist({Name="Features", Options={"Aimbot","ESP","Speed"}, CurrentSelected={"ESP"}, Flag="Features"})
main:CreateKeybind({Name="Ultimate Key", CurrentKeybind="F", Flag="UltKey"})
main:CreateColorPicker({Name="Highlight Color", Color=Color3.fromRGB(255,80,80)})
advanced:CreateButton({Name="Show Flags", Callback=function()
    SynergyUI:Notify("Master: "..tostring(window.Flags.Master:GetValue()),3)
end})
advanced:CreateButton({Name="Set Volume 80", Callback=function() window.Flags.Volume:SetValue(80) end})
advanced:CreateButton({Name="Save Config", Callback=function() window:SaveConfig() end})
advanced:CreateButton({Name="Destroy Window", Callback=function() window:Destroy() end})

SynergyUI:Notify("Demo Ready | Press RightShift",4,Color3.fromRGB(138,141,168))

Window API

:CreateTab(name, icon?) – Creates tab.
:SetAccent(color) – Changes accent color.
:SaveConfig(filename?) – Saves flags & position.
:LoadConfig(filename?) – Restores config.
:Destroy() – Removes UI.
.Flags – Table with :GetValue()/:SetValue().

Notifications

SynergyUI:Notify(message, duration, color, position)

SynergyUI:Notify("Connected", 3, Color3.fromRGB(138,141,168), "TopRight")

Programmatic Control

All flagged controls expose :GetValue() and :SetValue(val) in window.Flags.

window.Flags.Master:SetValue(true)
print(window.Flags.Volume:GetValue())
window.Flags.Features:SetValue({"Aimbot","ESP"})

Theme & Customization

Accent color can be set in CreateWindow or via window:SetAccent(). Deep customization by modifying source.

window:SetAccent(Color3.fromRGB(200,80,80))
