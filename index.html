SynergyUI

A clean, modern, and fully-featured UI library for Roblox. Built for performance, customization, and ease of use.

local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

One-line installation • Zero dependencies • Config saving • Flags support

Getting Started

Load the Library

local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

Create Your First Window

local window = SynergyUI:CreateWindow({
    Title = "My Hub",
    AccentColor = Color3.fromRGB(0, 255, 100),
    ConfigFile = "config.json",    -- enables auto‑save/load
    ToggleKey = Enum.KeyCode.RightShift,
    Parent = nil,                  -- optional, defaults to CoreGui
    CloseOnEscape = false          -- destroy window when pressing Esc
})

Window Configuration

Available Parameters

ParameterTypeDefaultDescription
Titlestring"Synergy Hub"Title shown in top bar
AccentColorColor3#00ff64Main theme color
ConfigFilestring""File name for auto‑save (requires writefile/readfile)
ToggleKeyEnum.KeyCodeRightShiftKey to show/hide UI
ParentInstanceCoreGuiParent screen gui
CloseOnEscapebooleanfalseIf true, pressing Esc destroys the window

All other visual aspects (sizes, colors, fonts) are pre‑defined in the library and follow a modern dark theme. The accent color can be changed later with window:SetAccent(color).

Tabs

window:CreateTab(name, iconUrl?) → returns a table with all control constructors.

local main = window:CreateTab("Main")
local settings = window:CreateTab("Settings", "rbxassetid://123456789")  -- optional icon

Each tab object contains methods: CreateLabel, CreateSeparator, CreateSection, CreateParagraph, CreateButton, CreateToggle, CreateSlider, CreateDropdown, CreateChecklist, CreateTextInput, CreateNumberInput, CreateKeybind, CreateColorPicker, CreateRadioGroup.

Controls Reference

Toggle Flag

Boolean switch with state saving.

ParameterTypeDescription
NamestringLabel text
CurrentValueboolInitial state (default false)
FlagstringUnique identifier for saving
CallbackfunctionReceives new state
TooltipstringHover tooltip
main:CreateToggle({
    Name = "ESP",
    CurrentValue = true,
    Flag = "ESP",
    Callback = function(state) print(state) end
})

Programmatic: window.Flags.ESP:GetValue() / :SetValue(false)

Slider Flag

Numeric slider with increment.

ParameterTypeDescription
Range{number, number}Min and max values
IncrementnumberStep value (default 1)
CurrentValuenumberInitial value
FlagstringAuto‑save identifier
CallbackfunctionReceives current value
main:CreateSlider({
    Name = "Speed",
    Range = {16, 250},
    Increment = 1,
    CurrentValue = 50,
    Flag = "WalkSpeed",
    Callback = function(v) print(v) end
})

Button

Clickable action button.

ParameterTypeDescription
NamestringButton text
CallbackfunctionExecuted on click
TooltipstringHover tooltip
main:CreateButton({
    Name = "Greet",
    Callback = function() SynergyUI:Notify("Hello!", 2) end
})

Dropdown Flag

Selectable list. Returns object with SetOptions/SetValue.

local dd = main:CreateDropdown({
    Name = "Mode",
    Options = {"Normal", "Aggressive"},
    Flag = "Mode",
    Callback = function(opt) print(opt) end
})
dd:SetOptions({"Normal", "Tactical", "Fast"})

Checklist Flag

Multi‑selection list. Methods SetSelected/GetSelected.

local cl = main:CreateChecklist({
    Name = "Features",
    Options = {"Aimbot", "ESP"},
    Flag = "Features",
    Callback = function(selected) print(table.concat(selected, ", ")) end
})
cl:SetSelected({"ESP"})

TextInput

Free text field.

main:CreateTextInput({
    Name = "Message",
    CurrentText = "",
    Callback = function(text) print(text) end
})

NumberInput

Numeric input field with validation.

main:CreateNumberInput({
    Name = "Amount",
    CurrentValue = 10,
    Callback = function(num) print(num) end
})

Keybind Flag

Key capture with press detection.

main:CreateKeybind({
    Name = "Ultimate",
    CurrentKeybind = "F",
    Flag = "UltKey",
    Callback = function(key) print("Key set to", key) end
})

ColorPicker

RGB color selector.

main:CreateColorPicker({
    Name = "ESP Color",
    Color = Color3.fromRGB(0,255,0),
    Callback = function(c) print(c.R, c.G, c.B) end
})

RadioGroup

Exclusive option buttons.

Paragraph

Rich text with optional image.

main:CreateParagraph({
    Title = "Info",
    Content = "Complete documentation",
    Image = "rbxassetid://..."   -- optional
})

Complete Example

-- SynergyUI Advanced Demo – Showcases every control with flags and programmatic access
local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

local window = SynergyUI:CreateWindow({
    Title = "SynergyUI • Complete Demo",
    AccentColor = Color3.fromRGB(0, 255, 100),
    ConfigFile = "demo_config.json",
    ToggleKey = Enum.KeyCode.RightShift,
    CloseOnEscape = true
})

-- Create two tabs
local main = window:CreateTab("Controls")
local advanced = window:CreateTab("Advanced")

-- ========== MAIN TAB ==========
main:CreateSection("Basic Inputs")

local toggle = main:CreateToggle({
    Name = "Master Switch",
    Flag = "Master",
    CurrentValue = false,
    Callback = function(val) print("[Toggle] Master =", val) end
})

main:CreateButton({
    Name = "Notify Me",
    Callback = function() SynergyUI:Notify("Button clicked!", 2, Color3.fromRGB(0,255,100)) end
})

main:CreateSlider({
    Name = "Volume",
    Range = {0, 100},
    Increment = 5,
    CurrentValue = 50,
    Flag = "Volume",
    Callback = function(v) print("[Slider] Volume =", v) end
})

local dropdown = main:CreateDropdown({
    Name = "Game Mode",
    Options = {"Casual", "Competitive", "Hardcore"},
    Flag = "GameMode",
    Callback = function(opt) print("[Dropdown] Mode =", opt) end
})

local checklist = main:CreateChecklist({
    Name = "Enabled Features",
    Options = {"Aimbot", "ESP", "Speed", "Fly"},
    CurrentSelected = {"ESP"},
    Flag = "Features",
    Callback = function(selected) print("[Checklist] Selected:", table.concat(selected, ", ")) end
})

main:CreateTextInput({
    Name = "Custom Message",
    CurrentText = "Hello",
    Callback = function(text) print("[TextInput] Text =", text) end
})

main:CreateNumberInput({
    Name = "WalkSpeed",
    CurrentValue = 16,
    Callback = function(num) print("[NumberInput] WalkSpeed =", num) end
})

main:CreateKeybind({
    Name = "Ultimate Key",
    CurrentKeybind = "F",
    Flag = "UltKey",
    Callback = function(key) print("[Keybind] New key =", key) end
})

main:CreateColorPicker({
    Name = "Highlight Color",
    Color = Color3.fromRGB(255, 80, 80),
    Callback = function(c) print("[ColorPicker] RGB:", c.R, c.G, c.B) end
})

main:CreateRadioGroup({
    Name = "Theme Preset",
    Options = {"Neon", "Classic", "Matrix"},
    CurrentValue = "Neon",
    Callback = function(opt) print("[Radio] Preset =", opt) end
})

main:CreateSeparator()
main:CreateParagraph({
    Title = "Information",
    Content = "This is a complete demo of SynergyUI. All controls support flags, auto-save, and programmatic access.",
    Image = "rbxassetid://123456789"  -- placeholder
})

-- ========== ADVANCED TAB ==========
advanced:CreateSection("Programmatic Control")
advanced:CreateLabel("Use window.Flags to get/set values")

advanced:CreateButton({
    Name = "Show Current Flags",
    Callback = function()
        local msg = string.format("Master: %s\nVolume: %s\nGameMode: %s\nFeatures: %s",
            tostring(window.Flags.Master:GetValue()),
            tostring(window.Flags.Volume:GetValue()),
            window.Flags.GameMode:GetValue(),
            table.concat(window.Flags.Features:GetValue(), ", ")
        )
        SynergyUI:Notify(msg, 4, Color3.fromRGB(0,255,100))
    end
})

advanced:CreateButton({
    Name = "Set Volume to 80",
    Callback = function()
        window.Flags.Volume:SetValue(80)
    end
})

advanced:CreateButton({
    Name = "Toggle Master Switch",
    Callback = function()
        local current = window.Flags.Master:GetValue()
        window.Flags.Master:SetValue(not current)
    end
})

advanced:CreateButton({
    Name = "Change GameMode to Hardcore",
    Callback = function()
        window.Flags.GameMode:SetValue("Hardcore")
    end
})

advanced:CreateButton({
    Name = "Enable ESP + Aimbot in Features",
    Callback = function()
        window.Flags.Features:SetValue({"ESP", "Aimbot"})
    end
})

advanced:CreateButton({
    Name = "Save Config",
    Callback = function()
        window:SaveConfig()
        SynergyUI:Notify("Configuration saved", 2)
    end
})

advanced:CreateButton({
    Name = "Load Config",
    Callback = function()
        window:LoadConfig()
        SynergyUI:Notify("Configuration loaded", 2)
    end
})

advanced:CreateButton({
    Name = "Change Accent Color",
    Callback = function()
        window:SetAccent(Color3.fromRGB(math.random(0,255), math.random(0,255), math.random(0,255)))
    end
})

advanced:CreateButton({
    Name = "Destroy Window",
    Callback = function()
        window:Destroy()
    end
})

SynergyUI:Notify("SynergyUI Demo Loaded | Press RightShift to toggle", 5, Color3.fromRGB(0,255,100))

Window API

:CreateTab(name, icon?) – Creates a new tab and returns control builder.
:SetAccent(color) – Dynamically changes accent color (updates stroke, title, tab highlight).
:SaveConfig(filename?) – Saves flags, window position and size, accent color.
:LoadConfig(filename?) – Restores saved configuration.
:Destroy() – Completely removes UI and cleans connections.
.Flags – Table containing all flagged controls with :GetValue() and :SetValue().

Notifications

SynergyUI:Notify(message, duration, color, position)

SynergyUI:Notify("Connection established", 3, Color3.fromRGB(0,255,100), "TopRight")
-- Positions: "TopRight", "TopLeft", "BottomRight", "BottomLeft"

Programmatic Control

All controls with a Flag are exposed in window.Flags with two methods:

  • :GetValue() – returns current value (bool, number, string, or table for checklist)
  • :SetValue(val) – updates the control and triggers its callback
-- Toggle
window.Flags.Master:SetValue(true)
print(window.Flags.Master:GetValue())

-- Slider
window.Flags.Volume:SetValue(75)

-- Dropdown (string)
window.Flags.GameMode:SetValue("Hardcore")

-- Checklist (table of selected options)
window.Flags.Features:SetValue({"Aimbot", "ESP"})

Dropdown and Checklist also return additional methods: SetOptions(newList), SetSelected(tbl), GetSelected().

Theme & Customization

SynergyUI comes with a sleek dark theme by default. You can only change the accent color through CreateWindow({AccentColor = ...}) or window:SetAccent(). All other visual aspects (sizes, fonts, background colors) are fixed to ensure a consistent look. If you need deeper customization, you can modify the source code directly.

Example of changing accent to red:

local window = SynergyUI:CreateWindow({
    Title = "Red Theme",
    AccentColor = Color3.fromRGB(255, 50, 50)
})
-- later
window:SetAccent(Color3.fromRGB(0, 200, 255))
