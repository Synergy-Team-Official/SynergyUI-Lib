LIBRARY · COMPLETE UI FRAMEWORK

SynergyUI

Modern, resizable, fully-featured UI library for Roblox executors. 15+ control types, flag system, rich notifications, keybinds, theme engine & automatic config saving.

local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

⚡ Drag / Resize • Toggle Key (RightShift) • Flags auto-linked • Config file persistence • 4 built‑in themes

Getting Started

Load Library & Create Window

local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

local window = SynergyUI:CreateWindow({
    Title = "My Advanced Hub",
    AccentColor = Color3.fromRGB(138, 141, 168),
    IconSet = "lucide",                 -- downloads icon pack, enables named icons in tabs
    ToggleKey = Enum.KeyCode.RightShift,
    CloseOnEscape = true,
    ConfigFile = "SynergyConfig.json",  -- automatically saves all flags, window position/size
    Parent = nil                        -- defaults to CoreGui or PlayerGui in Studio
})

IconSet downloads icon definitions from GitHub. Built‑in set: "lucide". After setting IconSet, you can use string names like "home", "settings", "info" in CreateTab. The ConfigFile saves every flagged control, window geometry and minimised state – loads automatically on next launch.

Tabs with Icons

window:CreateTab(name, iconName?) → returns a tab object with all control constructors. If you provided IconSet in CreateWindow, you can pass a string icon name (e.g., "info", "settings", "home", "sword"). Otherwise iconName is ignored.

local main = window:CreateTab("Combat", "sword")
local visuals = window:CreateTab("Visuals", "eye")
local infoTab = window:CreateTab("Info", "info")

If you omit the icon or don't set IconSet, tabs will be text‑only.

Complete Control Reference

Toggle Flag

Boolean switch with state & callback.

main:CreateToggle({
    Name = "ESP",
    CurrentValue = true,
    Flag = "ESP",
    Callback = function(state) print(state) end
})

Slider Flag

Numeric with increment, range, manual input.

main:CreateSlider({
    Name = "Speed",
    Range = {16, 250},
    Increment = 1,
    CurrentValue = 50,
    Flag = "WalkSpeed"
})

Button

Clickable action with optional tooltip.

main:CreateButton({
    Name = "Execute",
    Tooltip = "Runs the main script",
    Callback = function() SynergyUI:Notify("Done!", 2) end
})

Dropdown Flag

Single select, dynamic options, callback, multi‑select & searchable.

local dd = main:CreateDropdown({
    Name = "Mode",
    Options = {"Normal", "Aggressive"},
    CurrentOption = "Normal",
    MultiSelect = false,
    Searchable = true,
    Flag = "Mode"
})
dd:AddOption("Extreme")
dd:RemoveOption("Aggressive")

Checklist Flag

Multi‑selection list with selected count.

main:CreateChecklist({
    Name = "Features",
    Options = {"Aimbot", "Wallhack", "Speed"},
    CurrentSelected = {"Aimbot"},
    Flag = "Features"
})

TextInput Flag

Free text field with placeholder.

main:CreateTextInput({
    Name = "Message",
    CurrentText = "",
    Placeholder = "Type here...",
    Flag = "msg"
})

NumberInput Flag

Numerical input field (any number).

main:CreateNumberInput({
    Name = "Amount",
    CurrentValue = 10,
    Flag = "amount"
})

Keybind Flag

Capture key, calls callback on press & change.

main:CreateKeybind({
    Name = "Ultimate",
    CurrentKeybind = "F",
    Flag = "UltKey"
})

ColorPicker Flag

RGB slider picker, returns Color3. Includes rainbow toggle.

main:CreateColorPicker({
    Name = "ESP Color",
    Color = Color3.fromRGB(0,255,0),
    Flag = "espColor"
})

RadioGroup Flag

Exclusive option buttons.

main:CreateRadioGroup({
    Name = "Theme",
    Options = {"Dark", "Light"},
    CurrentValue = "Dark",
    Flag = "themeChoice"
})

Paragraph & Section

Informative blocks with optional image.

main:CreateSection("Information")
main:CreateParagraph({
    Title = "Info",
    Content = "Detailed description",
    Image = "rbxassetid://123"
})

Image (Expandable)

Collapsible image preview with description.

main:CreateImage({
    Title = "Preview",
    Image = "rbxassetid://123456",
    Description = "Optional caption"
})

Label & Separator

main:CreateLabel("Heading text")
main:CreateSeparator()

All Flag Controls – Extended Methods

Every control with a Flag is stored in window.Flags[flagName] and provides at least :GetValue() and :SetValue(val). Dropdown adds: AddOption(opt), RemoveOption(opt), ClearOptions(), Select(val) (for single) or Select(table) (multi). Checklist adds: AddOption(opt), RemoveOption(opt). ColorPicker returns Color3.

Window API & Methods

:CreateTab(name, iconName?) – Adds a new tab, returns tab object with control constructors. Icon name is only used if IconSet was provided.
:SetAccent(color) – Dynamically changes the accent color (borders, indicators, sliders).
:SetTheme(themeName) – Switches between built‑in themes: "Rise", "Dark", "Cyberpunk", "TokyoNight".
:Destroy() – Completely removes the UI and disconnects all events.
.Flags – Table containing all flagged controls (Toggle, Slider, Dropdown, Checklist, TextInput, NumberInput, Keybind, ColorPicker, RadioGroup). Each provides :GetValue() and :SetValue(val) plus specific methods.
.MainFrame – Direct access to the main GUI frame (advanced positioning).

Notification System

SynergyUI:Notify(options) – Queues and shows animated toasts. options can be a string (message) or a table with full control.

OptionTypeDescription
MessagestringNotification text
Typestring"info", "done", "error", "warning" (affects color & icon)
DurationnumberSeconds visible (default 4.2)
Positionstring"TopRight", "TopLeft", "BottomRight", "BottomLeft"
TypeColorColor3Custom left strip color (overrides Type)
CallbackfunctionCalled after notification closes
 
SynergyUI:Notify("Simple message")
SynergyUI:Notify({Message = "Success", Type = "done", Duration = 3, Position = "BottomRight"})
SynergyUI:Notify({Message = "Error!", Type = "error", Callback = function() print("closed") end})

Programmatic Control (Flags)

Every control with a Flag parameter becomes accessible via window.Flags[FlagName]. You can get/set values and modify options dynamically.

window.Flags.MasterToggle:SetValue(false)
local volume = window.Flags.Volume:GetValue()
window.Flags.Features:SetValue({"Aimbot", "ESP"})
window.Flags.GameMode:AddOption("Nightmare")
window.Flags.GameMode:RemoveOption("Casual")

Dropdown and Checklist also have :AddOption(opt), :RemoveOption(opt), :ClearOptions(). Keybind flag will execute its callback whenever the bound key is pressed (unfocused).

Theme & Customization

SynergyUI ships with 4 built‑in themes: Rise (default), Dark, Cyberpunk, TokyoNight. Change at runtime:

window:SetTheme("Cyberpunk")
window:SetAccent(Color3.fromRGB(255, 80, 80))

You can also modify the internal window.Theme table before creating controls to change sizes, padding, fonts, etc. The theme includes background, sidebar, element colors, corner radius, and typography.

UI Features: Drag via top bar, resize via bottom‑right handle, minimize/close buttons, toggle key (default RightShift), Escape can destroy.

Config File Persistence

Pass ConfigFile = "path.json" in CreateWindow to automatically save:

  • All flagged control values (including Color3)
  • Window position and size
  • Minimized state
  • Current theme (if changed via :SetTheme)

The file is written using writefile (if supported). The library saves 0.3 seconds after any change, and loads on next launch. No extra code required.

local window = SynergyUI:CreateWindow({
    Title = "Persistent UI",
    ConfigFile = "MyHubConfig.json"
})
