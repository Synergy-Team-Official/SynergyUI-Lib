SynergyUI

Modern, resizable, fully-featured UI library for Roblox executors. Includes 15+ control types, flag system, notifications, keybinds & theming.

local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

⚡ Drag / Resize • Toggle Key (RightShift) • Flags auto-linked • No external dependencies

Getting Started

Load Library & Create Window

local SynergyUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Synergy-Hub-Official/SynergyUI-Lib/refs/heads/main/SRC/source.lua"))()

local window = SynergyUI:CreateWindow({
    Title = "My Advanced Hub",
    AccentColor = Color3.fromRGB(138, 141, 168),
    IconSet = "lucide",
    ToggleKey = Enum.KeyCode.RightShift,
    CloseOnEscape = true,
    Parent = nil
})

IconSet downloads icon definitions from GitHub. Built‑in sets: "lucide". After setting IconSet, you can use string names like "info", "settings", "home" in CreateTab.

Window Configuration

Parameters (CreateWindow)

ParameterTypeDefaultDescription
Titlestring"Synergy Hub"Title shown in top bar
AccentColorColor3Color3.fromRGB(0,170,255)Main theme color
IconSetstringnilName of icon set (e.g. "lucide") – enables named icons in tabs
ToggleKeyEnum.KeyCodeRightShiftKey to show/hide UI
CloseOnEscapebooleanfalseIf true, pressing Escape destroys window
ParentInstanceCoreGui / PlayerGuiParent GUI container

After creation you can change accent color dynamically via window:SetAccent(Color3).

Tabs with Icons

window:CreateTab(name, iconName?) → returns a tab object with all control constructors. If you provided IconSet in CreateWindow, you can pass a string icon name (e.g., "info", "settings", "home"). Otherwise iconName is ignored.

local main = window:CreateTab("Combat", "sword")
local visuals = window:CreateTab("Visuals", "eye")
local infoTab = window:CreateTab("Info", "info")

If you omit the icon or don't set IconSet, tabs will be text‑only.

Complete Control Reference

Toggle Flag

Boolean switch with state & callback.

main:CreateToggle({
    Name = "ESP",
    CurrentValue = true,
    Flag = "ESP",
    Callback = function(state) print(state) end
})

Slider Flag

Numeric with increment, range, manual input.

main:CreateSlider({
    Name = "Speed",
    Range = {16, 250},
    Increment = 1,
    CurrentValue = 50,
    Flag = "WalkSpeed"
})

Button

Clickable action with optional tooltip.

main:CreateButton({
    Name = "Execute",
    Tooltip = "Runs the main script",
    Callback = function() SynergyUI:Notify("Done!", 2) end
})

Dropdown Flag

Single select, dynamic options, callback.

local dd = main:CreateDropdown({
    Name = "Mode",
    Options = {"Normal", "Aggressive"},
    CurrentOption = "Normal",
    Flag = "Mode"
})
dd:SetOptions({"New1","New2"})

Checklist Flag

Multi‑selection list with selected count.

main:CreateChecklist({
    Name = "Features",
    Options = {"Aimbot", "Wallhack", "Speed"},
    CurrentSelected = {"Aimbot"},
    Flag = "Features"
})

TextInput Flag

Free text field with placeholder.

main:CreateTextInput({
    Name = "Message",
    CurrentText = "",
    Placeholder = "Type here...",
    Flag = "msg"
})

NumberInput Flag

Numerical input field (any number).

main:CreateNumberInput({
    Name = "Amount",
    CurrentValue = 10,
    Flag = "amount"
})

Keybind Flag

Capture key, calls callback on press & change.

main:CreateKeybind({
    Name = "Ultimate",
    CurrentKeybind = "F",
    Flag = "UltKey"
})

ColorPicker Flag

RGB slider picker, returns Color3.

main:CreateColorPicker({
    Name = "ESP Color",
    Color = Color3.fromRGB(0,255,0),
    Flag = "espColor"
})

RadioGroup Flag

Exclusive option buttons.

main:CreateRadioGroup({
    Name = "Theme",
    Options = {"Dark", "Light"},
    CurrentValue = "Dark",
    Flag = "themeChoice"
})

Paragraph & Section

Informative blocks with optional image.

main:CreateSection("Information")
main:CreateParagraph({
    Title = "Info",
    Content = "Detailed description",
    Image = "rbxassetid://123"
})

Label & Separator

main:CreateLabel("Heading text")
main:CreateSeparator()

main:CreateSection("Selection Controls")

advanced:CreateSection("Input & Keybinds")

Window API & Methods

:CreateTab(name, iconName?) – Adds a new tab, returns tab object with control constructors. Icon name is only used if IconSet was provided.
:SetAccent(color) – Dynamically changes the accent color (borders, indicators, sliders).
:Destroy() – Completely removes the UI and disconnects all events.
.Flags – Table containing all flagged controls (Toggle, Slider, Dropdown, Checklist, TextInput, NumberInput, Keybind, ColorPicker, RadioGroup). Each provides :GetValue() and :SetValue(val).
.MainFrame – Direct access to the main GUI frame (advanced positioning).

Note: Dropdown flags also have :SetOptions(newOptions). Checklist flags have :SetSelected(table) and :GetSelected().

Notification System

SynergyUI:Notify(message, duration, color, position) – Queues and shows animated toasts.

ParameterTypeDescription
messagestringContent of notification
durationnumberSeconds visible (default 4.2)
colorColor3Left accent strip color (optional)
positionstring"TopRight","TopLeft","BottomRight","BottomLeft"
 
SynergyUI:Notify("Connection established", 3, Color3.fromRGB(0,255,0), "BottomRight")

Programmatic Control (Flags)

Every control with a Flag parameter becomes accessible via window.Flags[FlagName]. You can get/set values and modify options dynamically.

window.Flags.Master:SetValue(false)
local volume = window.Flags.Volume:GetValue()
window.Flags.Features:SetValue({"Aimbot", "ESP"})
window.Flags.GameMode:SetOptions({"Easy","Hard","Extreme"})

Checklist extra: :GetSelected() returns table of selected options.

Keybind flag will also execute the callback whenever the bound key is pressed (unfocused).

Theme & Customization

Set accent color during window creation or anytime via window:SetAccent(Color3). The theme includes dark background, smooth corners, and subtle strokes.

window:SetAccent(Color3.fromRGB(200, 80, 80))

You can also modify the internal window.Theme table before creating controls to change sizes, padding, fonts etc. (advanced).

UI Features: Drag via top bar, resize via bottom‑right handle, minimize/close buttons, toggle key (default RightShift), Escape can destroy.
